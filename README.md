# Course Management System

## Project Overview
A simplified course management system using Java to demonstrate OOP principles and data structures.
This is a Command-Line Interface (CLI) application.

## Team Roles
- **Frontend (UI)**: Whitney - `src/main/java/com/cms/ui`
- **Backend (Logic)**: Frankie - `src/main/java/com/cms/backend`

## Project Structure
- `src/main/java/com/cms/ui`: User Interface (CLI menus, input handling).
- `src/main/java/com/cms/backend`: Business logic and data persistence.

## Getting Started
1. Open the project in your IDE (VS Code, IntelliJ, Eclipse).
2. Run `src/main/java/com/cms/ui/Main.java`.
